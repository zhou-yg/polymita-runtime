# 数据

程序的底层资产，最常用的数据工具，如数据库，KV

数据的可以作以下分类

- 存储方式
 - 行式
 - 列式

- 数据结构
 - 键值对
 - 列（List）表（Table）
 - 树
 - 图

- 特殊类型
 - 多媒体
 - 图表

对于一个大型产品，只需要上述的几个类型数据的互相组合即可满足

## 存储方式

数据在磁盘上的存储方式不同，导致读写时的性能差异

行模式存储适合 OLTP

列模式存储适合 OLAP


## 数据结构

键值对是最基本的数据结构

表格是列表衍生出的一种特殊约束场景

列表对应编程语言的数组，描述的具备通用相似性的场景

树是在列表的基础加入了 父子 的层级关系，便于人类进一步阅读理解

图则是在树的基础上，加入了更多的关联关系
